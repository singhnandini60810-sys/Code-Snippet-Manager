# Code Snippet Manager

A simple browser-based application to store and manage reusable code snippets.

## Features

- Save code snippets
- Select programming language
- Search snippets
- Persistent storage using LocalStorage
- Clean developer-friendly UI

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage

## How To Run

Download repository.

Open:

index.html
